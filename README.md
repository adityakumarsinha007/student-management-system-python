# Student Management System

A menu-driven Student Management System developed using Python.

# Features

- Add Student
- View Student Details
- Search Student by Roll Number
- Update Student Details
- Delete Student Records
- Input Validation
- Exception Handling

# Concepts Used

- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Exception Handling
- Menu Driven Programming

# Technologies Used

- Python 3

# Time Complexity

| Operation | Complexity |
|----------|----------|
| Add Student | O(n) |
| View Student | O(n) |
| Search Student | O(n) |
| Update Student | O(n) |
| Delete Student | O(n) |

# How to Run

```bash
python main.py
```
